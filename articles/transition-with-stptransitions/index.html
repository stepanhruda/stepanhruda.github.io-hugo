Transition with STPTransitions

Putting the easy into easy transitioning

Stepan Hruda on 31 May 2014

iOS 7 SDK has been around for more than a year now. Among other things, UIKit now allows us to completely (well, almost, more on that later) customize transitions as well as make them interactively react to the user's finger gestures. Chances that you already wrote some inter-controller animation sweetness should be pretty high. Unless you live in Backwardcompatibilitown.

The catch

Of course, there has to be a catch. Honestly, the transitioning API is unnecessarily verbose and a bit too abstract. I'm not a huge fan.

On top of my complaining, I wrote a wrapper called STPTransitions that makes it easier to animate all of your wonderfully lickable controllers. We have been using it in production for several months now, and I am quite fond of the resulting architecture improvements and code cleanliness. STPTransitions now also allows you to make your transitions interactive in just a couple of lines of code.
